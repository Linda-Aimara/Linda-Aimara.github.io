2. Filtrar genes con cambios relevantes (logFC y expresin)

Aqu seleccionamos los genes ms representativos:

|logFC| 2 cambios grandes entre grupos

|logCPM| 2 expresin suficientemente alta

Recordatorio

Qu es logFC?

logFC = log fold change Es la medida que cuantifica cunto cambia la expresin de un gen entre dos condiciones, por ejemplo:

Neonatos vs Adultos

Tratado vs Control

El smbolo | | significa valor absoluto, es decir, ignorar el signo.

Un valor de: |log2(FC)| 2 se traduce a: Fold change 2^2 = 4 veces

Interpretacin biolgica

logFC +2

El gen est sobreexpresado:

Expresin 4 veces mayor en neonatos que en adultos

logFC 2

El gen est sobreexpresado:

Expresin 4 veces mayor en adults que en neonatos

# Cargar libreras para manipulacin de datos
library(tidyverse)
library(dplyr)

# Ordenamos por logFC y filtramos con criterios biolgicamente relevantes
lfc2 <- lfc %>%
  arrange(desc(logFC)) %>%                                # Ordenar por logFC (de mayor a menor)
  filter(abs(logFC) >= 2 & abs(logCPM) >= 2)               # Filtrar genes con alta diferencia y expresin

# Revisamos el resultado filtrado
View(lfc2)

Este paso reduce ruido y nos permite enfocarnos en los genes ms biolgicamente significativos.

3. Cargar matriz de expresin normalizada (CPM)

Para realizar un heatmap se necesitan valores de expresin por muestra, no solo los estadsticos del modelo

# Leer tabla de conteos normalizados en formato CPM (Counts Per Million)
expr <- read.table("dato_tsv/cpms_edgeR_adultos_vs_neonatos.txt")

# Exploramos la tabla
View(expr)

4. Seleccionar la expresin solo de los genes filtrados

Extraemos nicamente las filas correspondentes a los genes que seleccionamos previamente.

# Extraemos la matriz de expresin para los genes filtrados.
# Usamos rownames(expr) para compararlos con lfc2$gene
dat1 <- as.matrix(expr[rownames(expr) %in% lfc2$gene, ])

View(dat1)

5. Reordenar la matriz en el mismo orden que la tabla filtrada

Esto asegura que los genes aparezcan en el heatmap segn el orden del anlisis (til para interpretacin).

# match() permite reordenar los genes exactamente en el orden de lfc2$gene
dat2 <- as.matrix(dat1[match(lfc2$gene, rownames(dat1)), ])

View(dat2)

Reordenar es crucial para que la visualizacin sea coherente con la tabla de resultados.

6. Crear un Heatmap con la funcin base de R (heatmap)

# Paletas de colores para el heatmap
library(RColorBrewer)

# Crear heatmap base en R
figg_heatmap <- heatmap(
  dat2,
  Rowv = NA,                     # No agrupar genes (mantiene el orden)
  Colv = NA,                     # No agrupar columnas
  scale = "row",                 # Normalizar por fila (gen)
  main = "DE genes (heatmap base R)",
  col = brewer.pal(n = 8, name = "Blues")   # Paleta de colores azul
)

Este heatmap es simple, pero til para entender la estructura de los datos.
